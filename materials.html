Tools & Resources I Use

In my teaching journey, I utilize a variety of tools and resources to provide engaging, interactive, and effective learning experiences. These tools range from content creation platforms, coding environments, and classroom management systems to AI assistants like ChatGPT to enhance both learning and teaching outcomes.

Jupyter Notebook - An interactive environment for coding, data analysis, and visualization.

W3Schools - A comprehensive platform to learn web development, covering HTML, CSS, JavaScript, and more.

Kahoot - A gamified platform to make learning fun and interactive through quizzes and live sessions.

Google Classroom - A platform to manage classwork, assignments, and student communication in one place.

Arduino IDE - A coding environment for programming microcontrollers used in robotics and IoT projects.

Canva - A user-friendly graphic design tool to create visually appealing presentations, posters, and content.

ChatGPT - An AI assistant used for brainstorming ideas, generating content, and coding solutions.

LINE - A messaging platform used for quick announcements and class communication.

Codecademy - An interactive platform for students to learn coding in various languages through hands-on practice.

Notion - A versatile tool for organizing class notes, assignments, and collaborative projects in one place.
